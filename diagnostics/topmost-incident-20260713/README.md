# Topmost GPU incident evidence index

This directory contains bounded text/log exports collected after the 2026-07-13 display-stack
failure. Large crash dumps remain at their original paths and are indexed in `INCIDENT_REPORT.md`;
they are not copied into or committed with the repository.

Binary `.evtx` exports and copied runtime logs are local forensic evidence only. The three requested
Markdown deliverables contain the human-readable timeline, conclusions, and validation procedure.
